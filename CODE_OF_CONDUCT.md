# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/),
version 2.1.

In short: be respectful and constructive. Harassment, discrimination, and
personal attacks are not tolerated. We welcome contributions from everyone
regardless of background or experience level.

Report unacceptable behavior to the maintainer via the email in the repository's
commit history. Reports are handled confidentially.

The full text is available at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.
